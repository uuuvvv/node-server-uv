## 文件结构
```
|-- node-server-uv
    |-- .gitignore         //git忽略文件
    |-- app.js             //入口文件
    |-- package-lock.json
    |-- package.json       //依赖目录
    |-- README.md          //项目介绍文件
    |-- bin                //配置文件
    |   |-- www
    |-- public             //静态资源文件
    |   |-- images          //图片
    |   |-- javascripts      //js文件
    |   |-- stylesheets       //样式文件
    |       |-- style.css
    |-- routes              //路由文件
    |   |-- index.js
    |   |-- users.js
    |-- views               //页面文件
        |-- error.jade
        |-- index.jade
        |-- layout.jade

```
## 项目命令
- 项目启动
```
npm run start
npm start
```
- 依赖安装
```
npm install
```
